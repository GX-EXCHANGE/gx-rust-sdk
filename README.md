# gx-exchange-rust-sdk

SDK for GX Exchange API trading with Rust.

## Usage Examples

See `src/bin` for examples. You can run any example with `cargo run --bin [EXAMPLE]`.

## Installation

`cargo add gx-exchange_rust_sdk`
